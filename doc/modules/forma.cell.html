grow (prevp, domain, ruleset)
Cellular automata growth. Performs one CA tick on pattern prevp in the specified domain but only adding one cell at a time at the edge of an existing pattern

Parameters:

  • prevp the previous iteration of the pattern
  • domain the points in which the CA operates
  • ruleset a list of forma.rules for performing the CA on

Returns:

    the next iteration, and a bool specifying if convergence has been reached.
