Functions

grow.rules.genSig (...)
Signature generation. Converts a list of growth parameters into a string signature. Requres 10 inputs in the order specified in the ruleSpec.

Parameters:

  • ... provided integer growth parameters

Returns:

    string signature for use when generating rules
grow.rules.genRule (rulesig)
Ruleset generation. Generates a ruleset for pattern growth. Takes a string signature, either by hand or generated by genSig.

Parameters:

  • rulesig string signature specifying the ruleset

Returns:

    verified set of forma.grow.rules
grow.rules.checkGrowth (r)
Ruleset verification. Checks that a ruleset is valid (has sensible values).

Parameters:

  • r a forma.grow.rules table (as generated by genRule)
grow.rules.passRule (rule, nFour, nDiag, nEdge)
Growth rule. Checks that a set of tile parameters passes the specified ruleset.

Parameters:

  • rule a forma.grow.rules table to be checked against.
  • nFour the number of occupied cardinal directions to the tile/
  • nDiag the number of occupied diagonal directions to the tile/
  • nEdge the number of edge tiles neighbouring the tile.
growPattern (rule, points, domain, seeds, rng)
Pattern growth function

Parameters:

  • rule A forma.rule table specifying how the pattern should grow, alternatively a forma.rule generating string.
  • points Number of requested points
  • domain (optional) A forma.pattern specifying the limits of pattern growth
  • seeds (optional) Number of randomly positioned seed points (must be either nil or 1 for free growth)
  • rng (optional )A random number generating table, following the signature of math.random.

Returns:

    growth forma.pattern specified in domain
